Dijkstra's Algorithm Visualizer

WATCH THE FAMOUS PATH-FINDER IN ACTION

One of my favorite algorithms that we learned in class is Dijikstra's Algorithm. I was mind-blown by how the algorithm is able to find the closest path between two nodes and sought a way to visualize it for myself. This program uses the Pygame module to serve as a visualizer for the algorithm. View the source code here!

My first hurdle was figuring out what technology I would use to visualize this algorithm. After some research, I found that other's have used pygame as a way to create a grid interface and display the path as blocks traveled. I followed a tutorial by tech with tim to set up the GUI, and implemented the actual Dijkstra's algorithm on my own.

Implementing this in python was a unique challenge because I was used to writing this algorithm in Java. I found a way to represent a priority queue in Python and went from there. The GUI allows you to select a destination block, a source block, and put any form of black barricades in between. The program will find the most efficient way to get from the source block to the destination block by minimizing the amount of blocks traveled!
