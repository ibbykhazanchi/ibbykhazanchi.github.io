L1 & L2 Cache Simulator

CACHE SIMULATOR FOR MY COMPUTER ARCHITECTURE CLASS

This is a program I wrote for my Computer Architecture and Assembly class. We basically had to simulate an L1 and L2 cache using C. Our program had to represent sets, associativity, and blocks, and we also had to implement FIFO and LRU replacement policies.

I really liked this project because it was the grand-finale to the whole class. Also, our program had to satisfy thousands of test cases, so I really had to ensure I was writing robust code. Another layer of complexity was writing this in C, a very unforgiving language; thus, I had to be aware of C's memory management system. I used bitwise operations to efficiently compute tag bits and set bits for indexing memory addresses. I'm happy to say that I got a perfect score on this assignment!
